# Job Application Sample Analysis

This repository contains my sample data analysis website for a data analyst job application assignment.

## Project

The featured analysis predicts global video game sales using release information, review scores, genre, platform, and publisher group. The project includes data cleaning, exploratory data analysis, preprocessing pipelines, model comparison, random forest tuning, feature importance, limitations, and conclusions.

## Files

- `index.qmd` — portfolio landing page
- `analysis.qmd` — full sample data analysis
- `gamesales.csv` — video game sales dataset used in the analysis
- `resume.pdf` — resume
- `cover-letter.pdf` — cover letter
- `_quarto.yml` — Quarto website configuration
